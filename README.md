# web12_12A2_ngoclien_myportfolio
Chủ đề
Website portfolio cá nhân, giới thiệu bản thân, hoạt động ngoại khóa, thành tích và thông tin liên hệ.Trang được xây dựng bằng HTML, CSS, Bootstrap, JavaScript và triển khai qua GitHub Pages.
Thành viên: Hoàng Nguyễn Ngọc Liên – Lớp 12 – Làm việc cá nhân
Phân công công việc
Vì dự án được thực hiện cá nhân, toàn bộ các phần việc đều do tôi đảm nhận:
Phân tích template và xây dựng sitemap
Viết khung HTML và cấu trúc trang
Thiết kế CSS, màu sắc, card và giao diện
Tối ưu hình ảnh
Viết phần mô tả nội dung (About – Activities – Achievements)
Thêm JavaScript: Night Mode, dropdown menu, xử lý form
Kiểm thử responsive (mobile – tablet – laptop)
Deploy GitHub Pages
Viết báo cáo + README
/ (root)
├── index.html
├── style.css
├── main.js
├── /img
├── README.md
ách chạy dự án
1. Chạy website trên máy cá nhân (Local Run)
Tải repo hoặc clone bằng lệnh:
git clone <URL-repo-của-bạn>
Mở thư mục dự án.
Mở file:
index.html
Website sẽ chạy trực tiếp trên trình duyệt (Chrome / Safari / Edge).
Không cần cài đặt thêm vì dự án dùng HTML/CSS/JS thuần + CDN Bootstrap.
2. Chạy website online qua GitHub Pages (Deploy)
Nếu muốn chạy bản online:
Vercel: https://web12-12-a2-lien-myportfolio-duj4.vercel.app/
